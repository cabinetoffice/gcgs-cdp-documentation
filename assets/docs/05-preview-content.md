# Preview site content

Start edit → Edit content → Start edit navigation → Edit navigation → **Preview content** → Request review

This guide shows how to:
* Open your pull request
* Open the preview site
* Review your changes

## Step 1 - Open your pull request

Switch to the Pull requests tab and open the pull request you created earlier.
   
   <img src="../images/19-select-pull-request.png" width="450">

## Step 2 - Open the preview site
Locate the comment titled 'Preview deployed', then select the preview link.
   
   <img src="#">

## Step 3 - Review your changes
Review the preview site to confirm that your changes appear correctly

   <img src="#">

Update your content or navigation and preview again
